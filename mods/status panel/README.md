# Status Panel Mod v1.7.3

本版定位：正式体验整合版。

## 重点

- 基于最新主前端 index.html 重新合并状态栏 Hook。
- 主前端增加本地状态块隐藏兜底：状态栏脚本未 ready 时也先隐藏正文状态块。
- 状态栏脚本 ready 后，会再清理一次历史气泡里的残留状态块。
- 候选编辑改成更稳定的卡片内联编辑，输入时不再被刷新打断。
- 顶部“异常”文案改为“状态”，避免误解成程序错误。
- 自然语言地点识别增强：优先保留“资料室门口 / 餐厅门口 / 学校操场上”等完整地点。
- 自然语言装备候选更克制：普通“抱着/看了看平板”不再自动生成装备变更。
- 保留 v1.7.2 warmstyle 的浅色官方暖色风格。
