# 角色状态面板 Mod

这是一个用于 Xuqi_LLM / Fantareal 的状态栏 Mod。它会在聊天页显示一个小悬浮状态栏，用来记录角色的地点、HP、MP、身体状态、精神状态、装备、当前目标等信息。

它不是聊天页的大控制台，更像一个小便签：平时折叠成圆球，需要时展开看当前状态。

## 安装方式

把压缩包里的：

```text
mods/status panel/
```

覆盖到项目同名目录即可。

状态数据保存在项目根目录：

```text
data/status_panel/characters.json
```

所以后续覆盖 Mod 文件时，正常不会覆盖你的当前状态。

## 后台入口

浏览器打开：

```text
/mods/status-panel/app/
```

后台主要有这些页面：

```text
状态总览
角色编辑
字段设置
资料扫描
调试
```

## 主要功能

- 聊天页悬浮状态栏
- 后台角色状态编辑
- 字段显示顺序设置
- `status_panel` 初始化资料扫描
- `status_panel_update` 动态状态更新
- 当前状态持久化，退出重开后仍然保留
- aliases / 别名绑定
- 原记忆逐条扫描，生成候选状态
- 安全扫描、差异扫描、强制覆盖

## 三种数据来源

### 当前状态

这是状态栏真正显示的内容，保存在：

```text
data/status_panel/characters.json
```

聊天页优先读取这里。

### 初始化资料

使用 `status_panel`，负责第一次建立角色，或补空字段。它不应该反复覆盖游玩后的当前状态。

### 动态变化

使用 `status_panel_update`，负责记录本轮发生的状态变化。应用后会写入当前状态存档。

## aliases / 别名

角色可以设置别名，例如：

```text
Yucca
别名：小尤、尤加、Yuca
```

这样原记忆里写“小尤”，扫描时也能绑定回 Yucca。

注意：原记忆自然语言不会自动创建新角色，只会绑定已有角色名或已有 aliases。
