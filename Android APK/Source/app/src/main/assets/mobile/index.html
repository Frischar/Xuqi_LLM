世界书设置

命中规则与注入限制

“每轮最多注入条数”指的是单次回复里最多往 Prompt 里补多少条命中的世界书，不是世界书总词条数。

数值越小越克制，数值越大模型拿到的世界书上下文越多。

“任意命中”表示多个别名里命中一个就算；“全部命中”表示填的词都要同时出现。
