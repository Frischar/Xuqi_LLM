Frischar Project

欢迎来到 Xuqi Chat

这是一个面向长期陪伴聊天场景的本地 WebUI，支持角色卡、记忆库、世界书、差分立绘与独立存档。

当前版本由 Frischar 设计与整理，目标是把多角色陪伴聊天、可视化调教和本地化工作流整合到一个顺手的界面里。

进入聊天 打开配置

Features

你可以做什么

  • 按存档隔离角色卡、世界书、记忆和立绘
  • 导入或手动编辑角色卡
  • 把记忆库作为固定前情提要持续投喂给 AI
  • 管理立绘标签并切换对应表情

Workflow

推荐使用顺序

  1. 先在配置页填好聊天模型参数
  2. 在角色卡页导入或整理当前人设
  3. 在立绘页放好当前存档的表情图
  4. 在记忆页和世界书里补齐前情与设定
